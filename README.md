# coursetom-user-service
*User service for Coursetom app microservice*

Coursetom: My dummy app in online course system

## Services
- Student account management by admin role
- Authentication (Sign in/Sign up)
- Biodata (Student/Admin)
- Student course access (completed, owned, on-progress)

### Stacks
- Node.js + Express
- PostgreSQL (Sequelize ORM)
